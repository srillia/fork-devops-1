# ELK 日志收集


### 一、启动
```bash
docker-compose up -d
```

### 二、修改目录权限
> `${PWD}/elasticsearch`无访问权限会导致启动失败

```bash
chmod 777 ${PWD}/elasticsearch
```

### 三、在`logstash`中安装`json_lines`插件
```bash
# 进入logstash容器
docker-exec -it logstash /bin/bash

# 安装插件
cd /bin/
logstash-plugin install logstash-codec-json_lines

# 重启容器
docker restart logstash
```
